# EcoGestão - Controle de Coleta Reciclável

Este projeto consiste em um banco de dados relacional para gerenciar a coleta de materiais recicláveis em um condomínio ou comunidade. Foi desenvolvido como parte dos meus estudos no 2º semestre de Análise e Desenvolvimento de Sistemas.

### Objetivo
Demonstrar habilidades básicas em SQL, incluindo modelagem de dados, relacionamentos (PK/FK) e consultas com junções e filtros.

### Estrutura do Banco
O projeto conta com três tabelas principais:
- `moradores`: Cadastro de quem realiza as entregas.
- `tipos_materiais`: Catálogo de materiais aceitos e sua pontuação.
- `coletas`: Registro detalhado de cada movimentação.

### Tecnologias Utilizadas
- SQL (Structured Query Language)
- Banco de Dados Relacional (Ex: MySQL, PostgreSQL ou SQLite)

### Como usar
Você pode copiar o código do arquivo `script_banco_dados.sql` e executar em qualquer gerenciador de banco de dados para testar as tabelas e as consultas.
